# Security policy

## Supported versions

| Scope | Supported |
| ----- | --------- |
| Latest commit on `main` | Yes |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for undisclosed security problems.

Preferred: use **[GitHub Security Advisories](https://github.com/Blackphi6/video-speed-controller-plus/security/advisories/new)** for this repository so the report stays private until resolved.

If a weakness also affects the original **Video Speed Controller** extension, you may coordinate with upstream as appropriate — see [igrigorik/videospeed](https://github.com/igrigorik/videospeed).

## General bug reports

For non-security bugs and feature ideas, use [Issues](https://github.com/Blackphi6/video-speed-controller-plus/issues) on this repo (Plus-specific), or upstream if the behavior is not fork-specific.
